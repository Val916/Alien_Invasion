# Alien Invasion

This is the learning project from the great book 'Python Crash Course' by Eric Matthes.

## How to Play

### What You Need:
- A computer with Windows, Mac, or Linux
- Python 3.6 or higher (download from [python.org](https://www.python.org/downloads/))
- Internet connection (to download one library)

### Step-by-Step Instructions:

#### Step 1: Install Python
1. Go to [python.org](https://www.python.org/downloads/)
2. Download Python for your operating system
3. Run the installer and follow the prompts
4. **Important:** Check the box that says "Add Python to PATH" during installation
5. Click "Install Now"

#### Step 2: Download the Game
1. Download or copy this entire folder to your computer
2. Remember where you saved it

#### Step 3: Install the Game Library (Pygame)
1. Open Command Prompt (Windows) or Terminal (Mac/Linux)
2. Copy and paste this command, then press Enter:
   ```
   pip install pygame
   ```
3. Wait for it to finish installing (you'll see a "Successfully installed" message)

#### Step 4: Run the Game
1. Open Command Prompt (Windows) or Terminal (Mac/Linux)
2. Navigate to the game folder by typing:
   ```
   cd path/to/your/Alien_Invasion/folder
   ```
3. Type this command and press Enter:
   ```
   python alien_invasion.py
   ```
4. The game will launch in fullscreen!

### How to Play the Game:
- Click the **Play** button to start
- Use your **mouse** to move your ship left and right
- Click to **shoot bullets** at aliens
- Destroy all aliens to advance to the next level
- Don't let aliens reach the bottom!

### Need Help?
If you get an error:
- Make sure all files are in the same folder (alien.py, bullet.py, ship.py, etc.)
- Make sure the **images** folder is in the same location
- Try reinstalling pygame: `pip install --upgrade pygame`
